# 📚 Study Material Repository

## 📌 Overview

This repository is a centralized collection of structured study materials created during my learning process.

All resources are organized topic-wise and stored in a consistent format for easy access and reuse.

---

## 📂 Repository Structure

The repository follows a folder-based structure.

Each folder represents a specific topic and may contain:

- 📘 Study notes (PDF)
- 📄 Concept explanations
- ❓ Interview preparation material

Example:
/topic-name/
├── topic-notes.pdf
├── concepts.pdf
└── interview-questions.pdf


---

## 🧾 Naming Convention

Files are named in a consistent and descriptive manner:

- `<topic>-notes.pdf`
- `<topic>-concepts.pdf`
- `<topic>-interview-questions.pdf`

This ensures clarity and easy identification of content.
